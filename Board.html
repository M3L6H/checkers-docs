Board

Board()

Board class which holds the pieces and tracks their movement. Initializes with the red pieces on the first three rows on dark tiles and the black pieces on the last three rows on dark tiles.

Example
// Creates a new instance of the Board class
const board = new Board();

// Will Be a red piece
const redPiece = board.pieceAt(0, 1);

// Will be a black piece
const blackPiece = board.pieceAt(7, 0);

// Will be empty
const isEmpty = board.isEmpty(4, 4);
